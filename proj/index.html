• 1. Verificador de Paridade (Fácil)
    Crie uma função chamada eh_par(numero) que recebe um número inteiro. A função deve retornar True se o número for par e False caso contrário.
  • 2. Classificador de Idade (Fácil)
    Crie uma função chamada classifica_idade(idade) que recebe a idade de uma pessoa (inteiro) e retorna uma string com a sua classificação: "Criança" (menor de 12 anos), "Adolescente" (de 12 a 17 anos) ou "Adulto" (18 anos ou mais).
  • 3. Calculadora de Bônus Salarial (Fácil a Médio)
    Crie uma função chamada calcular_bonus(salario, anos_empresa). Se o funcionário tiver mais de 5 anos de empresa, ele recebe um bônus de 10% do salário. Caso contrário, recebe apenas 5%. A função deve retornar o valor do bônus.
  • 4. O Maior de Três Números (Médio)
    Crie uma função chamada encontrar_maior(a, b, c) que recebe três números inteiros ou floats e retorna o maior deles.
  • 5. Validador de Triângulos (Médio)
    Crie uma função chamada tipo_triangulo(lado1, lado2, lado3) que recebe o comprimento de três lados. Primeiro, a função deve verificar se esses lados podem formar um triângulo (a soma de dois lados deve ser sempre maior que o terceiro). Se não formarem, retorne "Não é um triângulo". Se formarem, retorne "Equilátero" (todos os lados iguais), "Isósceles" (dois lados iguais) ou "Escaleno" (todos os lados diferentes).
  • 6. Simulador de Caixa Eletrônico (Médio)
    Crie uma função chamada aprovar_saque(saldo, valor_saque). A função deve verificar se o valor_saque é menor ou igual ao saldo disponível e se o valor é múltiplo de 10 (já que o caixa só possui notas de 10, 20, 50 e 100). Retorne True se o saque for aprovado e False caso contrário.
